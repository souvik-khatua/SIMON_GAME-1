Demonstrating and Tags in HTML

In HTML, the <code> tag is used to display programming code in a monospaced font, while the <pre> tag preserves the formatting, including spaces and line breaks. The <var> tag is used to represent variables in mathematical or programming contexts.

Example: Displaying Raw Code in HTML


// This is a simple JavaScript function to add two numbers

    return a + b;
}

// Calling the function
let result = addNumbers(5, 10);
console.log("Sum:", result);

Understanding Variables

In programming, a variable is a symbolic name for a value that can change. For example, in JavaScript, we can define a variable like this: let x = 10;. Here, x is the variable holding the value 10.
